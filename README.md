# Instagram Post Page Recreation

## Project Overview
This project is a React-based recreation of an Instagram-style post page. The goal of this project is to demonstrate component-based architecture, reusable UI elements, and responsive design.

## Technologies Used
- React.js
- Vite
- CSS

## Component Structure
- Navbar
- Sidebar
- Feed
- PostCard
- Suggestions

## Reusable Components
The PostCard component is reusable and displays post data dynamically using props. The application uses an external data file to render multiple posts.

## Features
- Responsive layout for desktop, tablet, and mobile
- Reusable React components
- Static Instagram-inspired interface
- Dynamic rendering using JavaScript array and map()

## Functionality
The application displays a social media style page where users can view posts, profile details, comments, and suggested accounts.

## Conclusion
This project helped me understand how to break a webpage into smaller components, reuse components, pass props, and build responsive interfaces using React.
